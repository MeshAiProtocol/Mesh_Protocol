Testing iframe embedding

This page tests whether mesh1.vercel.app can be successfully embedded in an iframe without redirect loops.

If the iframe shows content (even an authentication page), the fix is working. If it shows a blank page or error, there may still be issues.

Expected behavior: The iframe should show either the chat interface (if authenticated) or an authentication required message with a button to open in a new tab.
